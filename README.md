# 🌱 AI Crop Recommendation System

## 📌 Overview

AI Crop Recommendation System is a full-stack web application that helps farmers make data-driven decisions by analyzing soil images, weather conditions, and nutrient values to recommend the best crops.

---

## 🚀 Features

* Soil Type Detection using CNN (Deep Learning)
* NPK Value Estimation
* Weather API Integration (Temperature, Humidity, Rainfall)
* Crop Recommendation using Machine Learning
* Smart Crop Rotation Logic
* User Authentication (Login/Signup)
* Admin Dashboard

---

## 🧠 Tech Stack

* Backend: Flask (Python)
* Frontend: HTML, Tailwind CSS, JavaScript
* Database: SQLite
* ML/DL: TensorFlow, Scikit-Learn

---

## 🔄 Workflow

1. User uploads soil image
2. CNN predicts soil type
3. Soil → NPK mapping
4. Weather API fetch
5. ML model predicts top crops
6. Crop rotation logic applied

---

## 📊 Models Used

* CNN (MobileNetV2) for soil classification
* ML model for crop recommendation

---

## 📂 Dataset

* Soil image dataset (6 classes)
* Crop recommendation dataset
* NPK soil dataset
